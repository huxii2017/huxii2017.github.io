Python Color Libraries

Python offers several great packages for working with colors in plots:

1. Matplotlib

Matplotlib includes many predefined color maps and color names. > üëâ Try other colormaps such as 'plasma', 'inferno', 'coolwarm','viridis', or 'Pastel1'.

import matplotlib.pyplot as plt
import numpy as np

# Show some of Matplotlib's built-in colormaps
gradient = np.linspace(0, 1, 256)
gradient = np.vstack((gradient, gradient))
for cmp in ['plasma', 'inferno', 'coolwarm', 'Pastel1','viridis']:
    plt.figure(figsize=(6, 1))
    plt.imshow(gradient, aspect='auto', cmap=plt.get_cmap(cmp))
    plt.title(f"Matplotlib's '{cmp}' colormap")
    plt.axis('off')
    plt.show()

2. Seaborn

Seaborn builds on Matplotlib and provides high-level color palettes.

import seaborn as sns

sns.color_palette("hls", 8) #circular color systems
sns.color_palette("Paired") #categorical Color Brewer palettes
#Sequential color palettes
sns.color_palette("rocket", as_cmap=True) #Perceptually uniform palettes
sns.color_palette("rocket") #Discrete vs. continuous mapping
sns.color_palette("cubehelix", as_cmap=True) #Sequential ‚Äúcubehelix‚Äù palettes
#Custom sequential palettes
sns.light_palette("seagreen", as_cmap=True)
sns.color_palette("Blues", as_cmap=True)

3. Plotly

Plotly‚Äôs interactive charts also support a wide variety of built-in and custom colors.

Python ‚Äì Build a Custom Colormap

We can easily turn a list of HEX codes into a colormap using Matplotlib.

You can replace the list with any colors you like.
For example:

import matplotlib.colors as mcolors

grey, red = '#BEBEBE', '#FF0000'
icmp = mcolors.LinearSegmentedColormap.from_list("", [grey, red])

You can use it like this:

import matplotlib.pyplot as plt
import matplotlib.colors as mcolors
import numpy as np

# define your favorite colors
colors = ['#BEBEBE','#fff4e6', '#FF0000']

# create a colormap from the list
icmp = mcolors.LinearSegmentedColormap.from_list("my_palette", colors)

# visualize it
gradient = np.linspace(0, 1, 50)
gradient = np.vstack((gradient, gradient))

plt.figure(figsize=(6, 1))
plt.imshow(gradient, aspect='auto', cmap=icmp)
plt.axis('off')
plt.title("My Custom Colormap", fontsize=10)
plt.show()

R ‚Äì Make a Simple Custom Palette

R provides an equally easy way with colorRampPalette().

%%R

# create a palette function
my_palette <- colorRampPalette(c('#BEBEBE','#fff4e6', '#FF0000'))

# generate 50 intermediate colors
cols <- my_palette(50)

# visualize
image(1:50, 1, as.matrix(1:50), col = cols, axes = FALSE,
    main = "My Custom Color Palette")
